[−][src]Enum holochain_net_connection::protocol::Protocol

pub enum Protocol {
    NamedBinary(NamedBinaryData),
    Json(JsonString),
    Ping(PingData),
    Pong(PongData),
    P2pReady,
}

Low-level interface spec for communicating with the p2p abstraction notice this is not Serializable or Deserializable rmp_serde doesn't serialize enums very well... it uses indexes and arrays which are not (easily) compatible with other endpoints we use to/from NamedBinaryData to provide our own serialization wrapper

Variants

NamedBinary(NamedBinaryData)

send/recv binary data / i.e. encryption, signature messages

Json(JsonString)

send/recv generic json as utf8 strings

Ping(PingData)

send/recv a Ping message (ipc protocol spec)

Pong(PongData)

send/recv a Pong message (ipc protocol spec)

P2pReady

we have connected / configured the connection, ready for messages
